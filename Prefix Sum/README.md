# 누적합 (Prefix Sum) 
## BOJ 20002 - 사과나무 

(https://www.acmicpc.net/problem/20002)


* (1,1)부터 (N,N)까지 총이익 입력받기
* (1,1) ~ (N,N)까지 각각의 '누적합' DP[x][y] 구하기
* 빅오 : O(N**2)
```
	for (int x = 1; x <= N; x++) {
		for (int y = 1; y <= N; y++) {
			cin >> map[x][y];
			DP[x][y] = DP[x - 1][y] + DP[x][y - 1] - DP[x - 1][y - 1] + map[x][y];
		}
	}
```
